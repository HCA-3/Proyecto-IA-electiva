# Carpeta de pruebas

Esta carpeta agrupa scripts de prueba y ejemplos de configuración que originalmente estaban en `scratch/`.

Archivos:

- `test_fpdf.py`: Prueba básica de generación de PDF y manejo de celdas.
- `test_fpdf_loop.py`: Prueba de bucle para creación de varias líneas en PDF.
- `test_fpdf_wrap.py`: Prueba de ajuste automático de texto (`multi_cell`).
- `test_fpdf_y.py`: Prueba de posicionamiento vertical en coordenadas Y.

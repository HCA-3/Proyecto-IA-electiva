# Documentación del Proyecto

Esta carpeta contiene material auxiliar y documentación de referencia para el proyecto.

## Archivos principales

- `Taller-Practico-Seleccion-de-Flujos-de-Trabajo-de-IA (1).pdf`
  - Documento académico relacionado con el proyecto.
- `pdf_content.txt`
  - Texto extraído del documento PDF.
- `pdf_content_utf8.txt`
  - Versión en UTF-8 del contenido extraído.

## Organización recomendada

- `scripts/`
  - Contiene utilidades y herramientas auxiliares, como el extractor de PDF.
- `tests/`
  - Contiene pruebas de comportamiento y archivos de experimentación que antes estaban en `scratch/`.

> Nota: Los archivos original(es) aún permanecen en el proyecto. Se recomienda moverlos manualmente a esta carpeta si se desea mantener una estructura más limpia.
